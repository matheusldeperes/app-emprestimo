# Instruções para adicionar a fonte Nasalization

A fonte Nasalization Rg é especificada no manual de identidade visual.

## Como adicionar a fonte:

1. Baixe a fonte Nasalization em: https://www.dafont.com/nasalization.font
2. Extraia o arquivo `nasalization-rg.ttf`
3. Coloque o arquivo na pasta `assets/nasalization-rg.ttf`

## Alternativa:

O código está configurado para usar automaticamente:
- Nasalization (se disponível em assets/)
- Arial Black (alternativa similar)
- Helvetica-Bold (fallback)

A fonte será carregada automaticamente quando disponível.
